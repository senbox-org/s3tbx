Slope Calculation

Processor Description

I/O Parameters Tab

Source Product Group

Used to select the Slope Calculation source product. Use the ... buttons to open a data product currently not opened in the Sentinel Toolbox.

Supported Source Products

The following source products are supported for Slope Calculation:

  • Any product which contain an elevation band, a geocoding, and which is supported by SNAP.
Target Product Group

Name: Used to specify the name of the target product.

Save to: Used to specify whether the target product should be saved to the file system. The combo box presents a list of file formats.

Open in SNAP: Used to specify whether the target product should be opened in the Sentinel Toolbox. When the target product is not saved, it is opened in the Sentinel Toolbox automatically.

The Processing Parameters

Elevation band name:
The name of the elevation band in the source product. Default is 'elevation'.

Copy elevation band:
If selected, the elevation band is copied into the target product. Default is 'false'.

The Processing Result

The Slope Calculation processor provides the following outputs:

slope:
The terrain slope for the given pixel in degrees, from zero (flat) to 90 degrees.

aspect:
Aspect of each pixel as angle between North direction and steepest slope, from zero to 360 degrees.

elevation_variance:
Variance of elevation over a 3x3 pixel window.
