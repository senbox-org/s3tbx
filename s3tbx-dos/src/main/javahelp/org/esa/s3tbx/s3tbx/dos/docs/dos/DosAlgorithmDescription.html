Slope Calculation

Algorithm Specification

Slope computation

If altn with n = 1..9 are the altitudes consecutively numbered in a 3x3 window around the considered pixel, the slope is computed as

b = (alt3 + 2 * alt6 + alt9 - alt1 - 2 * alt4 - alt7) / 8
c = (alt1 + 2 * alt2 + alt3 - alt7 - 2 * alt8 - alt9) / 8

SLOPE = atan(sqrt(b/X * b/X + c/X * c/X))

Aspect computation

If altn with n = 1..9 are the altitudes consecutively numbered in a 3x3 window around the considered pixel, the aspect is computed as

b = (alt3 + 2 * alt6 + alt9 - alt1 - 2 * alt4 - alt7) / 8
c = (alt1 + 2 * alt2 + alt3 - alt7 - 2 * alt8 - alt9) / 8

ASPECT = atan2(-b, -c)
