FLH/MCI Processor

The FLH / MCI Processor calculates the Fluorescence Line Height and the Maximum Chlorophyll Index by exploiting the height of a measurement over a specific baseline for MERIS and OLCI images. For S2 MCI there is a separate tool.

The Processor can also be used to define new line height products because both the baseline bands and the signal band are freely configurable.
