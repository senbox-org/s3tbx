SNAP Data Processors - OLCI Anomaly Flagging Processor Description

Processor Description

I/O Parameters Tab

Source Product Group

OLCI L1b product: Used to select the OLCI L1b source product. The source product can also be a 'compatible' OLCI L1b product, however, all OLCI radiances, solar fluxes, the per-detector wavelength (lambda0), altitude and solar zenith angle must be present and named the same as the original data. Use the ... button to open a data product currently not opened in the Sentinel Toolbox.

Target Product Group

Name: Used to specify the name of the target product.

Save as: Used to specify whether the target product should be saved to the file system. The combo box presents a list of file formats.

Open in SNAP: Used to specify whether the target product should be opened in the Sentinel Toolbox. When the target product is not saved, it is opened in the Sentinel Toolbox automatically.

The Processing Parameters

writeSlopeInformation
If set to true, additional variables containing the per-pixel slope information are written to te output. I.e. a variable "max_spectral_slope" containing the maximal value of the slope in the pixel spectrum and a variable "max_slope_band_index" containing the band index where the slope is maximal are added to the target.

The Processor Output

The processor provides a flag band "anomaly_flags" containing the flags detected during processing. It additionally copies all OLCI input data to the output.
