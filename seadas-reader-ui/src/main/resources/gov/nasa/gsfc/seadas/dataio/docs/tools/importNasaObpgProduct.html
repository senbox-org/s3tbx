Import NASA Ocean Data Products

The NASA Ocean Biology Processing Group (OBPG) provides a consistently processed set of ocean color data products from several missions, including HawkEye, PACE, MODIS (Aqua and Terra platforms), SeaWiFS, OCTS and CZCS. For more information on the data and for data access, please refer to the NASA Ocean Color Web (https://oceancolor.gsfc.nasa.gov) .

The OBPG is also responsible for the data processing for the Aquarius mision. For more information on the data and for data access, please refer to the NASA Aquarius Web site (https://aquarius.nasa.gov)

SeaDAS supports the all of the data products produced by the OBPG, including Level 1 through Level-3 Data and Browse imagery
