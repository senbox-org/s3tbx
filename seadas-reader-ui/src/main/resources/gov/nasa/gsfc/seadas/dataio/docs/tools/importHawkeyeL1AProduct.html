NASA Hawkeye Level-1A Product Reader

This HawkEye product reader is for the L1A data of the HawkEye [https://uncw.edu/socon/hawkeye.html] instrument operating on the SeaHawk cubesat. The files are in the NetCDF4 format. See also [https://oceancolor.gsfc.nasa.gov/data/hawkeye/summary]

The reader will attempt to identify the corresponding geolocation file to use for navigating the product. If one does not exist (or cannot be identified), the product will be loaded without any georeferencing capabilities.
