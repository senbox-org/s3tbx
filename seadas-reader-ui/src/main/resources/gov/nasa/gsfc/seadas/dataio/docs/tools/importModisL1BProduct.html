NASA MODIS Level-1B Product Reader

This SeaDAS product reader is for the L1B data of the Moderate Resolution Imaging Spectrometer [https://modis.gsfc.nasa.gov] (MODIS) instruments operating on both the Terra and Aqua spacecraft. The files are in the HDF-EOS format.

The reader will attempt to identify the corresponding geolocation file to use for navigating the product. If one does not exist (or cannont be identified), the product will be loaded without any georeferencing capabilities.

If geolocation is possible, the reader implements a correction for the "bow-tie" effect. This is only for navigation/reprojection purposes. The underlying data are not modified.
