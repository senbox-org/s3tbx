NASA GOCI Level-1B Product Reader

This GOCI L1B product reader is for the L1B data of the GOCI (Geostationary Ocean Color Imager) [https://oceancolor.gsfc.nasa.gov/data/goci/] instrument operating on the Communication, Ocean and Meteorological Satellite (COMS) spacecraft. The files are in the HDF-EOS5 format.

Note: the current GOCI L1B data has a problem such that the file cannot be read.
